Sistema di gestione delle prenotazioni delle aule, laboratori, sale conferenza etc..

== Description ==
Questo plugin permette di gestire le prenotazioni di spazi come aule, laboratori, sale conferenza etc.. 
Le prenotazioni possono essere realizzate solo dagli utenti registrati al sito
== Installation ==

Di seguito sono riportati i passi necessari per l'installazione del plugin. la procedura dettagliata la potete trovare sul sito dedicato al plugin http://plugin.sisviluppo.info


1. Scaricare il plugin dal repository di wordpress o dal sito di riferimento
2. Attivare il plugin dal menu Plugins
3. Inserire gli atti lato amministrazione
